Overview

OmniPass is an advanced All-Pass filter plugin designed for modern audio production. It maintains a completely flat magnitude response, without altering volumes or frequency balance, while generating frequency-dependent phase shifts. This makes it ideal for correcting complex phase alignments in multi-mic or multi-source recordings, aligning live sound speaker systems, or creating creative interference and dispersion effects.

It features capabilities such as different processing modes, flexible internal routing, phase offset functionality, and an intuitive graphical interface that allows you to visualize the filter's response in both phase and magnitude.

OmniPass es un plugin avanzado de filtro All-Pass pensado para la producción de audio moderna. Mantiene una respuesta de magnitud completamente plana, sin modificar volúmenes ni balance de frecuencias, pero genera desplazamientos de fase dependientes de la frecuencia, lo que lo hace ideal para corregir alineaciones complejas en grabaciones multipista, alinear sistemas de altavoces en vivo o crear efectos creativos de interferencia y dispersión.

Posee características tales como: diferentes modalidades de procesamiento, ruteo interno, offset de fase y una interfaz gráfica que permite visualizar la respuesta del nodo tanto en fase como en magnitud.
